STATES

- SUCCESS
    - expression succeeded

- END_OF_STREAM
    - no tokens left, but expression succeeded

- UNEXPECTED_SYMBOL
    - found unexpected token, not necessarily an error

- ERROR
    - expression failed




ALTERNATIVE

 - SUCCESS:
    - a child returns SUCCESS

 - END_OF_STREAM:
    - never

 - UNEXPECTED_SYMBOL:
    - never

 - ERROR
    - child returns END_OF_STREAM
    - no child returned SUCCESS




OPTIONAL

 - SUCCESS:
    - when no tokens left
    - child returns SUCCESS
    - child returns UNEXPECTED_SYMBOL
    - child returns END_OF_STREAM

 - END_OF_STREAM:
    - never

 - UNEXPECTED_SYMBOL:
    - never

 - ERROR
    - child returns error
    - child returns undefined state




REPETITION

 - SUCCESS:
    - child returns END_OF_STREAM
    - child returns UNEXPECTED_SYMBOL
    - reached end-of-stream

 - END_OF_STREAM:
    - never

 - UNEXPECTED_SYMBOL:
    - never

 - ERROR
    - child returns ERROR



RULE

 - SUCCESS:
    - child returns SUCCESS

 - END_OF_STREAM:
    - never

 - UNEXPECTED_SYMBOL:
    - never

 - ERROR
    - child returns UNEXPECTED_SYMBOL
    - child returns END_OF_STREAM
    - child returns ERROR
    - child returns undefined state




SEQUENCE

 - SUCCESS:
    - all expressions applied without problems

 - END_OF_STREAM:
    - never

 - UNEXPECTED_SYMBOL:
    - never

 - ERROR
    - child (not last) returns END_OF_STREAM
    - child returns UNEXPECTED_SYMBOL
    - child returns ERROR




TOKEN

 - SUCCESS:
    - when symbols match

 - END_OF_STREAM:
    - when no token left

 - UNEXPECTED_SYMBOL:
    - when symbols dont match

 - ERROR
    - never



SPAWNED BY

- SUCCESS
    - token

- END_OF_STREAM
    - alternative
    - optional
    - repetition
    - rule
    - sequence
    - token

- UNEXPECTED_SYMBOL
    token

- ERROR
    - alternative
